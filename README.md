# AAA Innovation Week - Intake Access

This repository serves as a collaboration space for the AAA team's Innovation Week project focused on intake access and authorization.

## Purpose

This repo is used to share ideas, designs, and prototypes between team members during the innovation week. It's a centralized place for:

- Design documents and proposals
- Architecture diagrams
- Proof of concepts
- Research findings
- Benchmark testing results
- Implementation notes

## Structure

```
.
├── benchmark/       # Performance benchmarks and testing results
│   ├── README.md
│   ├── benchmark_test.go

│   ├── analyze_results.py

│   └── FAIRNESS_ANALYSIS.md
├── design/          # Design documents and architecture proposals
│   └── spicedb_evaluation.md
└── README.md
```
